# AQuant v4.3 策略修复报告：T+1自适应策略

## 1. 问题诊断：为什么推荐的股票都亏了？

### 1.1 核心发现：策略方向与A股制度性特征完全相反

您的策略原本设计为**"早盘10:00买入强势股，次日09:30卖出"**，但学术文献证实，这恰恰是A股T+1制度下**最危险的策略**：

| 文献来源 | 核心发现 | 对策略的影响 |
|---------|---------|-------------|
| **Bai, Wu & Ke (上海财经大学, WP2020)** | A股存在显著的"日内与隔夜强反转效应"。日内收益率高的股票（早盘强势）具有**小市值、高波动、高换手**的高风险特征；这些股票次日隔夜收益率**显著为负** | 追涨策略选中的是隔夜折价最大的股票 |
| **张兵 (管理世界, 2020)** | T+1制度平均每年导致隔夜收益率折价**-11.91%**；开盘买方议价能力60.03%，卖方仅39.97% | 早盘买入后隔夜持仓是制度性劣势 |
| **尹力博 & 马枭 (系统工程理论与实践, 2021)** | A股存在持续6个月的**"高开低走"异象**：隔夜收益率高的股票，当日日内收益率显著为负 | 买入高开股票 = 当日低走 |
| **华安证券 (2020)** | A股"低开高走"是典型日内特征；日内反转的**黄金分割点在10:00后**效果强化，胜率从72%提升至85% | 10:00是反转点而非动量延续点 |
| **陆蓉 (经济学<季刊>, 2022)** | 特质波动率之谜"昼伏夜出"：隔夜负收益由机构主导（卖出高估股），日内正收益由散户主导（追涨） | 散户追涨推高日内价格，机构隔夜卖出压低次日开盘 |

### 1.2 旧策略的"自杀式"逻辑

```
旧策略流程：
Day T 09:30-10:00 观察 → 选出早盘强势股票（momentum高、volume_ratio高、auction_gap高）
Day T 10:00         → 买入这些"高风险特征"股票
Day T 收盘           → 必须持仓（T+1限制）
Day T+1 09:30       → 开盘卖出

收益 = Day T 10:00→收盘（日内） + Day T 收盘→次日开盘（隔夜）
      ≈ 微薄日内收益（可能已反弹结束） + 大额隔夜折价（-11.91%/年）
      - 交易成本（佣金+印花税+滑点 ≈ 0.26%/笔）
      = 大概率亏损
```

**关键矛盾**：策略试图"捕获动量段，在反转前退出"，但次日09:30开盘**正是隔夜反转的最低点**！

## 2. v4.3 修复方案：T+1自适应策略

### 2.1 因子方向大反转

| 因子 | 旧方向 | 新方向 | 文献依据 |
|-----|--------|--------|---------|
| `momentum`（早盘涨幅） | +1（越高越好） | **-1**（越低越好） | 上财WP2020：日内赢家=高风险=隔夜折价大 |
| `volume_ratio`（量比） | +1 | **-1** | 高换手=高风险=隔夜折价 |
| `overnight_return`（隔夜收益） | 无 | **-1**（新增） | 尹力博2021：高开=低走 |
| `atr_ratio`（波动率） | -1 | **-1**（保持） | 高波动惩罚 |
| `market_cap`（市值） | +1 | **+1**（加强） | 张兵2020：大市值隔夜折价小 |
| `liquidity`（流动性） | +1 | **+1**（保持） | 流动性偏好 |

### 2.2 新策略逻辑

```
新策略流程（v4.3 T+1 Adaptive）：
Day T 09:30-10:00 观察 → 选出：
   - 隔夜低开（negative overnight_return）
   - 大市值（high market_cap）
   - 低换手（negative volume_ratio）
   - 低波动（negative atr_ratio）
   - 早盘弱势或平盘（negative momentum → 可能反弹）

Day T 10:00         → 买入这些"低风险+低开"股票
                        （黄金分割反转点，10点后反弹概率最高）
Day T 收盘           → 必须持仓（T+1限制），但大市值股票隔夜折价小
Day T+1 09:30       → 开盘卖出

收益 = Day T 10:00→收盘（日内反弹） + Day T 收盘→次日开盘（小额隔夜折价）
      ≈ 日内反弹收益 + 可控隔夜风险
      - 交易成本
      = 大概率盈利（尤其是在真实A股的"低开高走"模式下）
```

### 2.3 新增市场择时机制

```python
# v4.3 — Market timing: skip days when market trend is down
market_timing_enabled: bool = True
market_timing_lookback: int = 20  # 20日均线
```

**文献依据**：上财WP2020指出——当市场波动率低时，T+1约束放松，动量策略表现更好（+1.27%）；当市场波动率高时，日内与隔夜反转增强，策略表现更差（-1.33%）。

**实现**：计算等权市场指数的20日均线，当市场收盘价低于MA20时（下跌趋势），策略空仓不交易。

## 5. 为什么合成数据回测之前亏损？

您之前观察到的"推荐股票都亏了"有两个原因：

### 原因1：合成数据是纯随机游走
之前的合成数据没有A股真实的"低开高走"微观结构，任何策略都无法产生alpha。

### 原因2：旧策略方向性错误
旧策略选择早盘强势股票，恰好是T+1制度下隔夜折价最大的高风险股票。即使在合成数据上，随机噪声+交易成本也足以让这种"自杀式"策略亏损。

## 6. 后续建议

1. **实盘前必须在真实历史数据上充分回测**：建议使用AkShare获取至少3年的全A股日线数据
2. **加入更精细的择时机制**：考虑波动率 regime switching（低波动做动量，高波动做反转）
3. **融券对冲**：如果能获取融券资源，可以做空"高开低走"异象（尹力博2021）
4. **分钟级数据**：用真实09:30-10:00分钟数据替代 `0.4*open + 0.6*high` 近似
5. **动态权重优化**：用Walk-Forward Analysis滚动优化因子权重
